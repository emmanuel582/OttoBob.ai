# OttoBob.ai — Otto University Platform

## Overview

**OttoBob.ai** is the central platform for managing students at **Otto University**. Students pay a $99 setup fee and $49/month, receive a financed Mac Mini, and leverage **Bob (the AI agent)** to help local non-tech businesses grow. 

Bob runs 95% of the operations automatically. OttoBob.ai captures a 5% perpetual network fee on increased revenue generated by these student agencies.

### Tech Stack

- **Frontend:** Next.js 16 (App Router) + TypeScript + Tailwind CSS v4
- **Backend:** Supabase (PostgreSQL + Auth + Storage)
- **Deployment:** Vercel

---

### Getting Started

#### 1. Install Dependencies

```bash
cd OttoBob.ai
npm install
```

#### 2. Set Up Environment

Create `.env.local` in the project root:

```
NEXT_PUBLIC_SUPABASE_URL=your-supabase-url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key
```

#### 3. Set Up Database

- Go to your **Supabase Dashboard → SQL Editor**
- Copy and paste the contents of `supabase-schema.sql`
- Click **Run** to create all tables, indexes, and policies

#### 4. Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

### Features (Phases 1 & 2)

- **Auth & Security** — Email & password login, session cookies, and route protection via Next.js Proxy Middleware.
- **Dashboard** — Enrollment overview with student counts by status.
- **Student Management** — Add, View, Edit, Delete students (CRUD).
- **Pipeline Status** — Applied → Contacted → Interviewing → Approved → Active → Rejected.
- **Search & Filter** — Search by name, major, email; filter by status.
- **Chronological Activity Log** — The core feature:
  - Add manual notes
  - Import iMessage conversations
  - Automatic system logs (status changes, student updates)
  - Automatic welcome email logs upon student creation (Bob AI)
- **Mission Control UI** — Dark theme (`#0a0a0f`), vibrant cyan (`#00e5ff`) accents, `144x144` (`h-36 w-36`) OB logo with `rounded-3xl` styling.
- **Mobile Responsive** — Works on phone, tablet, and desktop.
